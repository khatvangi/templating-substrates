# Templating Substrates v7 figure orchestration

## Goal

Generate all main-paper figure panels (Figs 1–6) for the templating substrates
manuscript v7 (PRX Life submission). Each figure is composed of multiple
panels; each panel is a separate PDF + PNG file generated by a separate
script. Composite assembly happens in Inkscape and is not the dispatch's
job.

## Working principle (do not violate)

**Two-layer figure workflow**: Boron generates panels. User composes panels
into composite figures in Inkscape. Never generate composite figures in
Python.

**One panel = one script = one output file pair (.pdf, .png)**. Do not
bundle panels into a single multi-axis matplotlib figure.

**No matplotlib defaults**. Use seaborn with paper context, ticks style,
and palette explicitly set per panel. If seaborn is unavailable, set
matplotlib rcParams to professional standards (font 8pt, axis_linewidth
1.0pt, savefig.dpi 600).

**Validation is mandatory**. Each script verifies its output (file exists,
PDF opens, PNG opens, fonts ≥ 7pt, no text overlap, file size < 5MB) before
declaring success.

use it. If not, the first script (fig5_panel_a_m0_m4_plateaus.py) should

```python
import seaborn as sns
import matplotlib.pyplot as plt

def apply_paper_style():
    sns.set_context("paper")
    sns.set_style("ticks")
    plt.rcParams.update({
        "font.size": 8,
        "axes.labelsize": 8,
        "xtick.labelsize": 7,
        "ytick.labelsize": 7,
        "axes.linewidth": 1.0,
        "lines.linewidth": 1.5,
        "savefig.dpi": 600,
        "savefig.bbox": "tight",
        "pdf.fonttype": 42,  # editable text in PDF
        "ps.fonttype": 42,
    })

OKABE_ITO = {

    "green": "#009E73",
    "vermilion": "#D55E00",
    "purple": "#CC79A7",
    "yellow": "#F0E442",
    "skyblue": "#56B4E9",
    "black": "#000000",
}
```

Subsequent scripts import and call apply_paper_style() at the top.

## What NOT to do

- Do not generate panel A of Fig 6 (the Beisson-Sonneborn schematic) — user
  draws this in Inkscape.
- Do not generate composite figures. One panel per script.
- Do not use matplotlib defaults. Either use the style module or explicitly
  set rcParams.
- Do not embed a literal copy of the existing diagnostic PNGs. Re-render with
  publication style.
- Do not use rainbow / jet / hsv colormaps anywhere.
- Do not pickle data into the panels (no PNG embedded inside PDF as raster).
